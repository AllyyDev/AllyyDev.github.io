Entwicklungsdatum: 2021

Erstes Projekt: Snakegame

Das ist mein "erstes" Code. Zwar habe ich vorher Informatikunterricht gehabt, Einstiegstutorials angeguckt, viel ausprobiert und geübt aber das ist mein erstes richtiges Spielprojekt, das ich mit Python (PyGame) entwickelt habe.

Das Spiel ist ein simples Snake game. Man hat eine Schlange, die links, rechts, oben und unten navigiert werden kann und den Äpfel(grün) isst. Wenn es einen isst, so wird sie um 1 Block größer. Wenn sie gegen sich oder die Wand schlägt, so stirbt sie und man verliert das Spiel. Das Spiel hat keine Ende.
